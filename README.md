## Project Overview

## How to Run

### Prerequisites
- Java 17+
- Maven 3.6+
- Docker (optional)

### Using Maven
\\\ash
mvn compile exec:java
\\\

### Using Docker
\\\ash
# 构建 Docker 镜像
docker build -t text-editor .

# 运行容器
docker run -it --rm text-editor

# 如果需要挂载本地目录
docker run -it --rm -v \E:\Maven\新建文件夹\massey-251-assignment1/data:/app/data text-editor
\\\
